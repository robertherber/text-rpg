# Dragon's Bane - A Fantasy Text RPG

A browser-based text adventure RPG with AI-generated images for each location.

## Features

- **Branching Story**: Make choices that affect your journey through a medieval fantasy world
- **Combat System**: Turn-based battles against goblins, wolves, skeletons, mages, and a dragon boss
- **Character Stats**: Health, strength, defense, magic, and leveling system
- **Inventory System**: Collect weapons, armor, potions, and key items
- **AI-Generated Images**: Each location features a unique image generated by DALL-E 3

## Getting Started

### Install Dependencies

```bash
bun install
```

### Configure OpenAI (Optional)

For AI-generated images, create a `.env` file with your OpenAI API key:

```bash
cp .env.example .env
# Edit .env and add your OPENAI_API_KEY
```

Without an API key, the game will use placeholder images.

### Run the Game

```bash
bun run index.ts
```

Then open http://localhost:3000 in your browser.

## How to Play

1. Enter your hero's name to begin
2. Read the story and choose your actions from the available options
3. Explore the village, talk to NPCs, and gather equipment
4. Find the dungeon key to access the dragon's lair
5. Defeat the dragon to win!

### Combat Controls

- **Attack**: Deal damage based on your strength
- **Defend**: Reduce incoming damage for one turn
- **Use Potion**: Restore health (requires health potion in inventory)
- **Flee**: Attempt to escape (50% success chance)

## Story Overview

The village of Millbrook is terrorized by an ancient dragon. You must:
1. Learn about the dragon from the tavern barkeep
2. Find the wizard Aldric in Whisperwood Forest to obtain the dungeon key
3. Optionally explore the goblin caves to retrieve the wizard's amulet
4. Storm the mountain fortress and defeat the dragon!
